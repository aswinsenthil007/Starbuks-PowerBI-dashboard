# Starbucks Power BI Dashboard

## Overview
Interactive Power BI dashboard for Starbucks beverage analysis.

## Features
- Total Beverages
- Average Calories
- Average Sugar
- Average Caffeine
- Beverage Category Distribution
- Top 5 Highest Caffeine Beverages
- Starbucks Global Presence

## Tools Used
- Power BI
- DAX
- Power Query
- CSV Dataset
